# Instagram Clone (Frontend)

This is the **React.js frontend** for the Instagram Clone project.  
It provides the user interface for interacting with posts, user profiles, authentication, likes, and comments.

 Features

- Responsive user interface for desktop and mobile  
- User authentication: signup and login pages  
- Display posts in a feed  
- Like and comment on posts  
- Search for other users and view profiles  
- Add, edit, and delete posts (frontend functionality)
  
-  Tech Stack
- React.js  
- HTML & CSS  
- JavaScript (ES6+)  
- Axios (for API requests)
